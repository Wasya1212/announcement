# announcement
Simple Announcement Website

Install the dependencies and devDependencies.

```sh
$ npm install
```

Build client side

```sh
$ npm run client-prod
```

Build server side

```sh
$ npm run server-prod
```

Dev client side

```sh
$ npm run client-dev
```

Dev server side

```sh
$ npm run server-dev
```

Build client&server

```sh
$ npm run prod
```

Dev client&server

```sh
$ npm run dev
```
